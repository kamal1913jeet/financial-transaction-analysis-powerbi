# financial-transaction-analysis-powerbi
This Power BI dashboard provides an interactive analysis of financial transactions and customer behavior. The project focuses on transforming raw transaction data into meaningful business insights through KPI monitoring, customer segmentation, transaction performance analysis, and geographical reporting.
# Finance Analysis Dashboard | Power BI

## Project Overview
This Power BI dashboard provides real-time insights into financial transactions and customer behavior.

## Key Features
- KPI Tracking
  - Total Amount
  - Total Transactions
  - Average Transaction Value
  - Fees and Tax Analysis

- Customer Segmentation
  - Retail
  - Premium
  - SME
  - Corporate

- Geographic Analysis
  - State-wise Tax Distribution

- Transaction Analysis
  - Transfer
  - Loan EMI
  - Deposit
  - Investment

- Interactive Filters
  - Year
  - Occupation
  - Category

## Tools Used
- Power BI
- DAX
- Power Query
- Excel/CSV
